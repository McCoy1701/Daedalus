int d_ArrayResize(dArray_t* array, size_t new_capacity)
Resize the internal data buffer of a dynamic array.
Parameters
  • array A pointer to the dynamic array whose internal buffer is to be resized.
  • new_size_in_bytes The desired new total size of the internal data buffer in bytes.
Returns
0 on success, 1 on failure.
Note
If `new_size_in_bytes` is smaller than the current allocated size, the data buffer is truncated. If the current `count` of elements exceeds the new capacity, `count` will be adjusted down to match the new capacity, effectively truncating the array's contents. @warning This function only affects the internal data buffer (`array->data`). The `dArray_t* array` pointer itself remains valid and points to the same `dArray_t` structure. Pointers obtained previously via `d_IndexDataFromArray` (or similar direct access) will become invalid if the underlying `array->data` buffer is reallocated and moved. Example: `d_ArrayResize(myArray, 10 * sizeof(int));` This resizes the internal buffer of `myArray` to accommodate 10 integer elements. If `myArray` previously held more than 10 elements, its `count` will be truncated.
